手动部署 fabric 网络

  • 1、通过 crypto-config.yaml 配置文件，生成所需要的证书文件:

    • 生成证书

      此时， 文件夹 crypto-config 里会有 ordererOrganizations 与 peerOrganizations 两个目录，分别存放 orderer 节点证书和 organization 节点证书。

    • cryto-config.yaml 参数说明：

      Name 和 Domain: 指定该组织的名字和域名，生成的证书内将包含该信息。
      Template.Count=2: 表明生成 2 套证书，一套是 peer0.org2 的，还有一套是 peer1.org2 的
      Users.Count=1: 指定该组织有多少普通 User（用户）（注：Admin 不计入 count 中）
